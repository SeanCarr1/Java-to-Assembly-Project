# Midterm Project

This folder contains the source files for the Microprocessors lab midterm project.

## Project Requirements

- Create 5 original Java programs
- Convert each Java program to assembly language with comments
- Cover these topics:
	- Selection Statement
	- Loop / Nested Loop
	- Procedure
	- Array
- Record a video explaining the Java and assembly versions
- Submit a document with the programs and their output
